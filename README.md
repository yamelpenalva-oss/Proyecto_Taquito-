## Descripción del proyecto

El presente proyecto consiste en el desarrollo de un robot futbolista autónomo, diseñado para participar en ligas nacionales de robótica.

El sistema integra hardware y software para permitir un comportamiento autónomo capaz de interactuar con el entorno en tiempo real, mediante la detección de la pelota, posicionamiento en el campo de juego y toma de decisiones.

El control del robot se realiza a través de microcontroladores ESP32 programados , aplicando principios de programación embebida.

---

## Objetivos

- Desarrollar un robot autónomo orientado a competencias de fútbol robótico  
- Integrar sensores, actuadores y microcontroladores  
- Implementar lógica de control en tiempo real  
- Aplicar conocimientos de electrónica y programación  
- Lograr un sistema funcional y eficiente  

---

## Materiales utilizados

- Motores de corriente continua (x4)  
- Módulo WiFi/Bluetooth  
- Microcontroladores ESP32 (x2)  
- Controladores de motor (TN6612 o TA6586)  

---

## Tecnologías y herramientas

- Lenguaje: C++ (framework Arduino / ESP32)  
- Entorno de desarrollo: Arduino IDE  
- Comunicación: WiFi y Bluetooth  
- Control de motores mediante PWM  
- Lectura de sensores digitales y analógicos  

---

## Lógica del sistema

El funcionamiento del robot se basa en una arquitectura de control que incluye:

- Lectura continua de sensores  
- Procesamiento de datos en tiempo real  
- Toma de decisiones mediante estructuras condicionales  
- Control de actuadores en función del entorno  

El comportamiento general contempla estrategias de búsqueda, ataque y corrección de trayectoria.

---

## Funcionamiento

El firmware desarrollado permite:

- Controlar el movimiento del robot  
- Procesar información del entorno  
- Ejecutar estrategias de juego  
- Adaptarse a condiciones variables durante la competencia  

El código se encuentra estructurado de forma modular, facilitando su mantenimiento y escalabilidad.
